Code, Source, Language: 사람이 하는 일, 원인 → Application, App, Program, Webpage: 컴퓨터가 하는 일, 결과 HTML은 웹페이지를 만드는 언어, 쉽고 중요하다. Web은 Public Domain으로 완전히 무료이며 전세계 공통 웹브라우저와 에디터를 준비한다. HTML 에디터로 아톰 추천

바탕화면에 프로젝트 폴더 WEB을 만들고 Open Folder (Ctrl+Shift+O)로 해당 폴더 잡아주면 프로젝트에 WEB이 뜸 폴더에서 우클릭해서 New File 클릭: 파일이름을 1.html로 하자. hello web 치고 해당파일 열어보면 나옴 웹페이지 열어서 우클릭 페이지 소스보기 (Ctrl+u)하면 html 소스를 볼 수 있음, h1 테그: heading, heading의 단계는 1~6 150개 이상의 태그가 있는데, 25개 정도의 tag가 평균적으로 쓰인다고 함, 중요도 순으로 태그를 배우면 되겠음
